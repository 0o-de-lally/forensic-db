# Source Code (`src/`)

Core logic for the `forensic-db` ETL system.

## Purpose
This directory contains the Rust implementation for extracting, transforming, and loading Libra blockchain archives into Neo4j.

## Subdirectories
- [analytics/](analytics/README.md): Data analysis and reporting tools.

## Key Files
- `lib.rs`: Library entry point.
- `main.rs`: Binary entry point.
- `load.rs`: Core loading logic.
- `extract_transactions.rs`: Logic for parsing transaction backups.
